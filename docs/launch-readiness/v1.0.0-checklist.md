## Section D — Manual smoke test

Manual happy-path verification on both URLs in all three locales. No CI equivalent for all steps.

**Canonical URL:** `https://iq-me.org/`

### D1 — Happy-path flow (per locale)

For each locale (EN / RU / PL) on the canonical site:

1. Open the URL in browser
2. Verify landing/consent scene renders correctly in the locale; copy is not EN fallback
3. Read validity envelope; confirm "Not today" exit path visible
4. Accept consent; confirm FR12: Continue control not available before reading disclosure
5. Complete 16 items; verify progress indicator (FR3) updates correctly
6. Submit final item; verify FR13 pre-result beat renders
7. View result: confirm score panel renders co-equal triplet (percentile, IQ-scale, uncertainty band) — **no single element visually dominates** (FR18)
8. Verify correct tail-scene renders for result band (bottom decile / mid-band / top decile)
9. Click a number on the result page → confirm methodology page opens in same locale (FR21)
10. Navigate methodology corpus; confirm `hreflang` links switch correctly between EN/RU/PL
11. Verify cite-this-page widget produces a valid APA-format citation
12. Verify no share button, no badge, no `navigator.share()` affordance (FR25)
