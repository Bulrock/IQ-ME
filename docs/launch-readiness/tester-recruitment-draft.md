## Purpose

IQ-ME needs a 5×3 native-speaker tester cohort (5 testers per language × 3 languages = 15 total) to validate the experiential credibility of the v1.0.0 build before launch. Testers run the test in their language, then post a credibility verdict on a dedicated GitHub Discussions thread.

The launch gate (Risk #12) requires **≥12/15 overall AND ≥4/5 per language** — no single-language failure masked by aggregation.

---

## What testers are asked to do

1. **Run the test** — complete the self-paced IQ-ME assessment in your native language (EN/RU/PL) at the public URL (TBD).
2. **Post a credibility verdict** — reply to the designated GitHub Discussions thread with:
   - "Credible" or "Not credible"
   - A brief reason (1–3 sentences): what felt right or wrong about the result framing, methodology explanations, or harm-mitigation copy.
3. **No other commitment** — one-time participation, no follow-up required unless the maintainer has a clarification question.

---

## Privacy and data posture

- **No email required.** Testers identify only by their GitHub handle.
- **No telemetry** — IQ-ME does not collect any data from test-takers; no tracking, no analytics, no cookies beyond session storage.
- **No personal data committed.** Only your GitHub handle (as chosen by you) appears in the credibility report.
- **Consent to handle being listed** — by posting on the GitHub Discussions thread, testers consent to their GitHub handle appearing in `docs/launch-readiness/tester-credibility-report.md`.

---

## Compensation

**Volunteer — no compensation.** This is an open-source project. There is no payment for participation.

---

## Archetype-balance target per language

For each language (EN, RU, PL), we aim for at least:

- **≥1 bottom-decile tester** — likely to receive the harm-mitigation / silent-companion-line copy; credibility of that copy is critical.
- **≥1 top-decile tester** — likely to receive the high-score framing; credibility of contextualisation ("this is one test, one day") matters.
- **≥1 skeptic** — someone who finds IQ-adjacent claims implausible; credibility of the methodology explanations and caveats is their primary lens.

Testers self-select; the archetype-balance note is informational, not a screening criterion. We will not know a tester's archetype until they run the test and self-report.
