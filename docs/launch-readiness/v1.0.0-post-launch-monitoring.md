## Monitoring checklist (per checkpoint)

For each checkpoint: open the canonical URL + mirror URL in a browser, run
through the quick smoke test below, check GitHub Discussions for any user
reports, verify external archival is still resolving.

**Quick smoke test (5 minutes per checkpoint):**
1. Open canonical URL → consent scene loads in EN
2. Open mirror URL → consent scene loads in EN
3. Complete one test session (or skip to result) → score renders
4. Click a score number → methodology page opens
5. Zenodo DOI resolves: `curl -I https://doi.org/<doi>` (or browser)
6. `gh run list --workflow scheduled.yml --limit 1` → most recent health-check green
